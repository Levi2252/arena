Arena+ Bukkit Plugin

The Arena+ bukkit plugin is to make arena's more fun for server players. Create areas for the arena's and build your own design for it. You can make mob arenas, spleef arenas, or pvp arenas. When you use Arena+ you get simple and easy-to-use commands.

Command List:

Command Description Permission
/arena join [arena_name] Lets a player join an arena if the user has the right permission. arenaplus.join
/arena list Lists all the arenas on the server. arenaplus.list
/arena create [arena_type] [arena_name] Creates an arena for a specific type with a custom name. arenaplus.create
/arena setarea [arena_name] Sets an area for the arena by left clicking for the first block of the arena, and right clicking for the second. arenaplus.create
/arena delete [arena_name] Deletes an arena and all the area claimed for the arena land. arenaplus.delete
/arena info [arena_name] Tells you what type of arena it is, and where the area for the arena is. arenaplus.info
